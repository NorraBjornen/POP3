Описание
    -Данное приложение является реализацией POP3 клиента для извлечения писем электронной почты.
    Приложение позволяет авторизироваться на сервере почты, извлекать сообщения, просмотривать заголовки,
    просмотривать содержимое тела сообщения, просматривать письмо полность, а также сохранять письмо в файл.
    -При запуске скрипта в качестве параметров указываются данные для подключению к серверу и авторизации.
    -После того как пользователь авторизовался, ему выводится информация о количестве писем в ящике, номера
    писем в ящике упорядочены от старых к новым (самое старое - номер 1), и будет предложено получить письмо по номеру.
    После того как письмо получено, пользователю выведется краткая информация о письме (число видов содержимого,
    наличие вложений), и приложение перейдет в режим обработки команд пользователя. Команды описаны ниже.


Состав:
    mail.py - главный скрипт, соединяющийся с сервером почты и обрабатывающий команды клиента

Требования
    Python версии 3


Использование приложения

    Справка:
            python mail.py
            python mail.py -h

    Параметры запуска:
            python mail.py [доменное имя почтового сервера] [порт] [имя пользователя] [пароль]

    Пример запуска:
            python mail.py pop.mail.ru 995 user@mail.ru password

    Команды пользователя:
            s                       - показать письмо полностью и сохранить в файл                                  (show)
            s h                     - показать заголовки письма                                                     (show headers)
            s c                     - показать сообщение письма                                                     (show content)
            h [название заголовка]  - показать значение указанного заголовка                                        (header n)
            c [номер содержимого]   - показать содержимое по указанному номеру (без заголовков типов сообщений)     (content n)
            t [количество строк]    - показать заголовки и указанное количество строк сообщения (Аналог TOP)        (top n)
            q                       - завершение сессии и выход                                                     (quit)
